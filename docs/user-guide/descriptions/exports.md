# Exports

You can export the structure of a description in the following formats:
- **PDF**
- **DOCX**
- **XML**

:::info
- File formats `PDF`, `DOCX` exports are done by [file transformer services](optional-services/file-transformers.md)
- You can use the export feature if current description status has this action. More details can be found [here](admin-guide/system-configuration/description-statuses.md)
:::
