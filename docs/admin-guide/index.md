# Admin Guide

This section covers all administrative features of OpenCDMP, including user management, content configuration, and system settings.

## Getting Started

New to OpenCDMP administration? Start with the [Admin Getting Started](../admin-guide/getting-started.md) guide to learn the basics of setting up and managing your OpenCDMP instance.

## Administrative Features

Administrators can manage and configure various aspects of the application:

- **Content Management**: Blueprints, templates, reference types
- **User Management**: Users, tenants, roles and permissions
- **System Configuration**: Statuses, workflows, notifications, languages
- **Monitoring**: Maintenance
