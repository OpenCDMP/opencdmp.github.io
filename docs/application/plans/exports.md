# Exports

You can export the structure of a plan in the following formats:
- **PDF**
- **DOCX**
- **JSON**
- **XML**

:::info
- `JSON` file is produced according to [RDA specifications](https://github.com/RDA-DMP-Common/RDA-DMP-Common-Standard)
- File formats `PDF`, `DOCX`, `JSON` exports are done by [file transformer services](/docs/supplementary-services/file-transformers.md)
- You can use the export feature if current plan status has this action. More details can be found [here](/docs/administration/plan-statuses.md)
:::
