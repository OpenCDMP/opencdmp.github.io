# Exports

You can export the structure of a description in the following formats:
- **PDF**
- **DOCX**
- **XML**

:::info
- File formats `PDF`, `DOCX` exports are done by [file transformer services](/docs/supplementary-services/file-transformers.md)
- You can use the export feature if current description status has this action. More details can be found [here](/docs/administration/description-statuses.md)
:::

:::tip

- Export can be used in [`Description Overview`](/docs/application/descriptions/description-overview.md)
- Export is also available in [Public Descriptions](/docs/application/descriptions/my-descriptions.md)

:::
