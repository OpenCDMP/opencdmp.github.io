# OpenCDMP Documentation

This repository contains the official documentation for **OpenCDMP**, built using **Docusaurus**. The documentation provides a comprehensive guide to using the OpenCDMP platform, covering topics such as platform setup, usage, file transformations, repository deposits, and API references. It serves as the central resource for developers, administrators, and users of the OpenCDMP platform.

## Overview

The OpenCDMP documentation is designed to be easy to navigate and understand, offering detailed explanations and examples for all aspects of the platform. It is hosted as a **Docusaurus** project, allowing it to be deployed as a static website for public access.
